# netty-pubsub

#### 介绍
百万级高性能分布式发布订阅系统，基于nio,netty开发，无锁并发，充分发挥cpu极限,
该项目已自带心跳检测功能，断线自动重连，非常稳定。
注意该项目软件编码为GBK，导入的时候一定要看清项目编码格式。
